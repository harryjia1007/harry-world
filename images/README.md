# 照片放這裡

把照片檔案放進這個資料夾，**檔名要完全對到下面的表**（英文小寫、.jpg），存檔案、commit、push 之後網站就會自動顯示，不用改任何程式碼。沒放檔案的地方會維持深色像素風佔位圖，不會壞版面。

建議每張照片先裁成接近下面標的比例再放進來，畫面會最好看（比例不完全準也沒關係，會自動裁切置中）。

| 檔名 | 用在哪裡 | 建議比例 |
|---|---|---|
| `work-notchglass.jpg` | WORK — NotchGlass 卡片 | 16:10（橫的截圖） |
| `work-injury.jpg` | WORK — 運動傷害預防系統卡片 | 16:10（橫的示意圖） |
| `work-next.jpg` | WORK — 更多正在路上（可留空） | 16:10 |
| `journey-aiii.jpg` | JOURNEY — Aiii.Ai 實習 | 4:3（直的照片也可以，會裁切） |
| `journey-book.jpg` | JOURNEY — 數位發展部專書收錄 | 4:3 |
| `journey-japan.jpg` | JOURNEY — 日本交流 | 4:3 |
| `journey-wave.jpg` | JOURNEY — AI WAVE SHOW 演講 | 4:3 |
| `journey-diplomat.jpg` | JOURNEY — 外交小尖兵 | 4:3 |
| `journey-biostem.jpg` | JOURNEY — BioSTEM 競賽 | 4:3 |
| `about.jpg` | ABOUT — 個人照片（拍立得風格） | 4:3，建議用比較清楚的正面照 |
| `strip-1.jpg` ~ `strip-5.jpg` | ABOUT — 底部生活照片底片，隨你放什麼日常照片 | 4:3 |

放好之後跟我說一聲，我幫你 commit + push，Cloudflare 會自動重新部署上線。
